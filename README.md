## Akira’s Portfolio Page

A simple HTML portfolio website introducing Akira, showcasing personal branding, web-development progress, and projects built while learning front-end development.
The page includes an introduction, a custom profile logo, resource links, and a list of practice projects.

# Features

- Personal introduction section

- Why Web Development? section

- Learning resources with external links

- Showcase of projects (Bookstore, Recipe Page, Travel Agency)

- tutorials from YouTube I watched

- Fully built using pure HTML

# Purpose

This project is part of my journey in learning front-end development.
It serves as a personal practice portfolio, helping me:

- Practice organizing content in HTML

- Display projects I’ve completed

- Build a foundation before adding CSS and JavaScript

- Track my growth as a beginner developer
